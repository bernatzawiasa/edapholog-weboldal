Használat
---------

1) Helyi megnyitás
   - Nyisd meg a `index.html` fájlt egy böngészőben.

2) GitHub Pages (ingyen)
   - Hozz létre egy új repót (pl. `edapholog-site`).
   - Másold be a fenti struktúrát a repo gyökérébe.
   - GitHub → Settings → Pages → Source: Deploy from a branch → `main`/`root`.
   - 1–2 perc múlva él az oldal.
   - A domainedet (edapholog.com) a Squarespace DNS-ben CNAME rekorddal irányítsd a GitHub Pages aldomainre.

3) Netlify (ingyen)
   - Netlify → Add new site → Deploy manually.
   - Drag & drop: a `edapholog-site` mappa tartalmát húzd be.
   - Domain settings alatt kösd rá az edapholog.com-ot (CNAME).
